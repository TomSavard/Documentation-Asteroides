\doxysubsection{Detailed Description}
Classe représentant l\textquotesingle{}écran de fin de jeu multijoueur. 

Cette classe gère l\textquotesingle{}affichage de l\textquotesingle{}écran de fin de jeu multijoueur, incluant le message de fin de partie, le nom du joueur gagnant, ainsi que les instructions pour redémarrer ou quitter le jeu.
